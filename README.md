# Realtime-dashboard
Realtime Dashboard using React, Redux, Socket.io, D3js, Express.

## Demo
![ScreenShot](https://github.com/amitmotgi/realtime-dashboard/blob/master/real-time-dashboard.gif)

## Installation
```
npm install
```

## Running the demo
```
npm run start
http://localhost:3000
```
